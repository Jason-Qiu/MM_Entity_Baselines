# minigpt4

## Setup


```bash
conda create -n minigpt4 python=3.10

sudo apt install git-lfs

git clone https://github.com/catid/minigpt4
cd minigpt4

git lfs install
git submodule update --init --recursive

conda activate minigpt4
pip install -r requirements.txt
```
